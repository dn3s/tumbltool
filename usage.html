Initial Setup

  1. Install TumblTool (obviously)

  2. Create A Directory For Your Theme

    If you're just going to be previewing your theme locally, put this directory wherever it's convenient for you, just like any other project.

    If you have set up a web server and want to preview your theme on one or more separate machines, you'll probably want your directory somewhere your web server can access (like under your DocumentRoot).

  3. Work On Your Theme File

    Hack away!

  4. Preview Your Theme

    Run tumbltool for a detailed list of options. Try tumbltool -t (your_theme_file) -o
