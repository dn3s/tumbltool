Contributing

There are three main ways to help.

  1. Report Bugs

    Bugs (usually) don't get fixed until someone reports them. If you run into issues, let me know!

  2. Create Example Content

    I will explain how to do that sometime soon.

  3. Implement Some Variables

    I am just about done cleaning up the interface for this and then it'll be easy to contribute in this way. Hang tight!

TumblTool is written by dn3s (github). Bugs, issues, and feature requests can be reported here. For other tumbltool-related inquiries you can contact me at tumbltool@devonsawatzky.ca, and for other inquiries you can contact d@devonsawatzky.ca.
